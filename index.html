TipKitchen App (UX)

Using a wide range of UX skills, I designed an app which connects users to professional chefs via video calling. The app would allow users to access professional advice in real time to fix any gastronomic problem. Users can also book workshops with their selected kitchen professional.

In order to create something useful, I conducted user interviews, established personas and user journeys, created user flows and sitemaps before designing low and mid-fidelity wireframes and an interactive prototype. Following moderated usability testing, I designed high-fidelity wireframes and an interactive high-fidelity prototype along with style guides to hand-off to developers.
